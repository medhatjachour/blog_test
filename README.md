## Getting Started

- clone the repo
```bash
git clone'https://github.com/medhatjachour/blog_test.git'
```

-install dependencies
```bash
npm init 
```

-run server 
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
